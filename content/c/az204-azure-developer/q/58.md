Service Bus integrates with Azure Functions or Logic Apps to process messages and store them in Blob Storage, and it scales easily as new stores open."

You work as a Cloud Integration Engineer for MDFT Pro, a well-known training agency that delivers certification courses to students worldwide. Claire, the Retail Solutions Lead, is building a point-of-sale (POS) data collection system for a hands-on Azure training lab. The system collects transaction data from 2,000 training store locations worldwide, with each store running 1 to 5 POS devices. Each device generates approximately 2 MB of transaction data every 24 hours including course material purchases, certification exam vouchers, and student enrollment fees. The collected data must be stored in Azure Blob Storage for analytics and reporting, with data correlated by device identifier to track individual device performance and transaction patterns. The solution must scale to accommodate additional stores opening as MDFT Pro expands its global training presence.

You need to implement a messaging solution to receive the POS device data.

**Solution:** Provision an Azure Service Bus. Configure a topic to receive the device data by using a correlation filter.

Does the solution meet the requirements?
